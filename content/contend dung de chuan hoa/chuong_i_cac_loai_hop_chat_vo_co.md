> Quy ước chuẩn hoá: công thức text thuần, không LaTeX; điều kiện phản ứng ghi trong ngoặc sau phương trình.

## I. SƠ ĐỒ PHÂN LOẠI CÁC CHẤT VÔ CƠ

Chất được chia thành hai nhóm lớn: đơn chất và hợp chất.

- Đơn chất:
  - Khí hiếm: He, Ne, Ar,...
  - Kim loại: Fe, Cu, Al, Na,...
  - Phi kim: O2, Cl2, S, P,...
- Hợp chất vô cơ:
  - Oxit: oxit bazơ, oxit axit, oxit lưỡng tính, oxit không tạo muối.
  - Axit: axit không có oxi, axit có oxi.
  - Bazơ: bazơ tan (kiềm), bazơ không tan.
  - Muối: muối trung hoà, muối axit.

## II. CÁC KHÁI NIỆM CƠ BẢN

### 1. Oxit

- Định nghĩa: oxit là hợp chất gồm hai nguyên tố, trong đó có một nguyên tố là oxi.
- Công thức tổng quát: MxOy (M là nguyên tố kim loại hoặc phi kim).
- Tên gọi: tên nguyên tố (kèm hoá trị nếu kim loại có nhiều hoá trị) + "oxit".
- Phân loại:
  - Oxit bazơ: thường là oxit của kim loại, tác dụng với dung dịch axit tạo muối và nước (CaO, CuO, Fe2O3,...).
  - Oxit axit: thường là oxit của phi kim, tác dụng với dung dịch bazơ tạo muối và nước (SO2, CO2, P2O5,...).
  - Oxit lưỡng tính: tác dụng với cả dung dịch axit và dung dịch bazơ tạo muối và nước (Al2O3, ZnO,...).
  - Oxit không tạo muối (oxit trung tính): không tác dụng với axit, bazơ, nước ở điều kiện thường (CO, NO,...).

### 2. Axit

- Định nghĩa: axit là hợp chất mà phân tử gồm một hay nhiều nguyên tử hiđro liên kết với gốc axit. Các nguyên tử hiđro này có thể bị thay thế bởi nguyên tử kim loại.
- Công thức tổng quát: HnA (A là gốc axit, n là số nguyên tử hiđro, bằng hoá trị của gốc axit).

Bảng 1.1. Một số axit và gốc axit thông thường:

| Kí hiệu gốc axit | Tên gốc axit | Hoá trị gốc axit | Axit tương ứng | Tên axit |
| --- | --- | --- | --- | --- |
| -Cl | clorua | I | HCl | axit clohiđric |
| =S | sunfua | II | H2S | axit sunfuhiđric |
| -NO3 | nitrat | I | HNO3 | axit nitric |
| =SO4 | sunfat | II | H2SO4 | axit sunfuric |
| =SO3 | sunfit | II | H2SO3 | axit sunfurơ |
| -HSO4 | hiđrosunfat | I | H2SO4 | axit sunfuric |
| -HSO3 | hiđrosunfit | I | H2SO3 | axit sunfurơ |
| =CO3 | cacbonat | II | H2CO3 | axit cacbonic |
| -HCO3 | hiđrocacbonat | I | H2CO3 | axit cacbonic |
| ≡PO4 | photphat | III | H3PO4 | axit photphoric |
| =HPO4 | hiđrophotphat | II | H3PO4 | axit photphoric |
| -H2PO4 | đihiđrophotphat | I | H3PO4 | axit photphoric |
| -OOCCH3 | axetat | I | CH3COOH | axit axetic |

- Phân loại và cách gọi tên:
  - Axit không có oxi: tên axit = axit + tên phi kim + "hiđric". Ví dụ: HCl — axit clohiđric; H2S — axit sunfuhiđric.
  - Axit có nhiều oxi: tên axit = axit + tên phi kim + "ic". Ví dụ: H2SO4 — axit sunfuric; HNO3 — axit nitric.
  - Axit có ít oxi: tên axit = axit + tên phi kim + "ơ". Ví dụ: H2SO3 — axit sunfurơ; HNO2 — axit nitrơ.

### 3. Bazơ

- Định nghĩa: bazơ là hợp chất mà phân tử gồm một nguyên tử kim loại liên kết với một hay nhiều nhóm hiđroxit (-OH).
- Công thức tổng quát: M(OH)n (M là kim loại, n là hoá trị của kim loại).
- Phân loại:
  - Bazơ tan (kiềm): NaOH, KOH, Ba(OH)2, Ca(OH)2.
  - Bazơ không tan: Cu(OH)2, Fe(OH)3, Al(OH)3,...
- Tên gọi: tên bazơ = tên kim loại (kèm hoá trị nếu kim loại có nhiều hoá trị) + hiđroxit.
  - Ví dụ: Cu(OH)2 — đồng(II) hiđroxit; Fe(OH)3 — sắt(III) hiđroxit.

### 4. Muối

- Định nghĩa: muối là hợp chất mà phân tử gồm nguyên tử kim loại (hoặc nhóm amoni NH4) liên kết với gốc axit.
- Công thức tổng quát: MnAm (M là kim loại hoá trị m, A là gốc axit hoá trị n).
- Phân loại:
  - Muối trung hoà: trong gốc axit không còn nguyên tử hiđro có thể thay thế bằng kim loại (NaCl, CaCO3, Na2SO4,...).
  - Muối axit: trong gốc axit vẫn còn nguyên tử hiđro chưa được thay thế bằng kim loại (NaHCO3, KHSO4, Ca(H2PO4)2,...).
- Tên gọi: tên muối = tên kim loại (kèm hoá trị) + tên gốc axit.
  - Ví dụ: CaCO3 — canxi cacbonat; Na2HPO4 — natri hiđrophotphat; Fe2(SO4)3 — sắt(III) sunfat.

## III. MA TRẬN TÍNH CHẤT HOÁ HỌC CỦA CÁC CHẤT VÔ CƠ

Bảng tóm tắt sản phẩm phản ứng giữa các loại chất (số trong ngoặc trỏ tới thí dụ bên dưới):

| Chất | Kim loại | Phi kim | Oxit axit | Oxit bazơ | Axit | Bazơ | Muối | H2O |
| --- | --- | --- | --- | --- | --- | --- | --- | --- |
| Kim loại | | Muối (1) | | | Muối + H2 (8) | | Muối + kim loại (12) | Kiềm + H2 (15) |
| Phi kim | Muối (1) | | | | | Muối (2) | Muối + phi kim (3) | Axit (4) |
| Oxit axit | | | | Muối (5) | | Muối + H2O (6) | | Axit (7) |
| Oxit bazơ | | | Muối (5) | | Muối + H2O (9) | | | Kiềm (16) |
| Axit | Muối + H2 (8) | | | Muối + H2O (9) | | Muối + H2O (10) | Muối + axit mới (11) | |
| Bazơ | | | Muối + H2O (6) | | Muối + H2O (10) | | Muối + bazơ mới (13) | |
| Muối | Muối + kim loại (12) | | | | Muối + axit mới (11) | Muối + bazơ mới (13) | 2 muối mới (14) | |

Thí dụ minh hoạ và điều kiện phản ứng (theo số thứ tự trong bảng):

1. 2Fe + 3Cl2 -> 2FeCl3 (t°)
2. Cl2 + 2NaOH -> NaCl + NaClO + H2O
3. Cl2 + 2NaBr -> 2NaCl + Br2 (phi kim mạnh đẩy phi kim yếu hơn ra khỏi muối)
4. Cl2 + H2O ⇌ HCl + HClO
5. CO2 + CaO -> CaCO3
6. CO2 + 2NaOH -> Na2CO3 + H2O, hoặc CO2 + NaOH -> NaHCO3 (tuỳ tỉ lệ mol)
7. SO3 + H2O -> H2SO4 (oxit axit tương ứng với axit tan)
8. 2HCl + Fe -> FeCl2 + H2↑ (kim loại hoạt động, đứng trước H)
9. 2HCl + CuO -> CuCl2 + H2O
10. H2SO4 + 2NaOH -> Na2SO4 + 2H2O, hoặc H2SO4 + NaOH -> NaHSO4 + H2O (tuỳ tỉ lệ mol axit và kiềm)
11. 2HCl + CaCO3 -> CaCl2 + CO2↑ + H2O (sản phẩm có chất khí hoặc kết tủa)
12. Cu + 2AgNO3 -> Cu(NO3)2 + 2Ag↓ (kim loại đứng trước đẩy kim loại đứng sau ra khỏi dung dịch muối)
13. 2KOH + MgSO4 -> Mg(OH)2↓ + K2SO4 (bazơ sản phẩm không tan)
14. Na2CO3 + CaCl2 -> CaCO3↓ + 2NaCl (hai muối tham gia phải tan, sản phẩm có kết tủa)
15. 2K + 2H2O -> 2KOH + H2↑ (kim loại kiềm, kiềm thổ mạnh tác dụng trực tiếp với nước)
16. Na2O + H2O -> 2NaOH (oxit bazơ tương ứng với bazơ tan)

### Một số phản ứng cần lưu ý

a) Oxit:

- 3CO + Fe2O3 -> 2Fe + 3CO2 (t°)
- 2HgO -> 2Hg + O2 (t°)
- CuO + H2 -> Cu + H2O (t°)
- Al2O3 + 6HCl -> 2AlCl3 + 3H2O
- Al2O3 + 2NaOH -> 2NaAlO2 + H2O (oxit lưỡng tính tác dụng với kiềm)

b) Bazơ:

- Cu(OH)2 -> CuO + H2O (t°)
- 4Fe(OH)2 + O2 + 2H2O -> 4Fe(OH)3 (kết tủa trắng xanh hoá nâu đỏ trong không khí)
- KOH + KHSO4 -> K2SO4 + H2O
- 4NaOH + Mg(HCO3)2 -> Mg(OH)2↓ + 2Na2CO3 + 2H2O
- Al(OH)3 + 3HCl -> AlCl3 + 3H2O
- Al(OH)3 + NaOH -> NaAlO2 + 2H2O (hiđroxit lưỡng tính)

c) Axit:

- Al và Fe không phản ứng với H2SO4 đặc, nguội và HNO3 đặc, nguội (bị thụ động hoá).
- Cu + 2H2SO4 (đặc, nóng) -> CuSO4 + SO2↑ + 2H2O
- Fe + 4HNO3 (loãng) -> Fe(NO3)3 + NO↑ + 2H2O

d) Muối:

- CaCO3 -> CaO + CO2↑ (t°)
- 2NaHCO3 -> Na2CO3 + H2O + CO2↑ (t°)
- NaHCO3 + NaOH -> Na2CO3 + H2O
- Fe + 2FeCl3 -> 3FeCl2
- Cu + Fe2(SO4)3 -> CuSO4 + 2FeSO4

## IV. PHƯƠNG PHÁP ĐIỀU CHẾ CÁC CHẤT VÔ CƠ

### 1. Điều chế oxit

- Kim loại + oxi: 4Al + 3O2 -> 2Al2O3 (t°)
- Phi kim + oxi: 4P + 5O2 -> 2P2O5 (t°)
- Oxi hoá quặng sunfua: 4FeS2 + 11O2 -> 2Fe2O3 + 8SO2↑ (t°)
- Nhiệt phân muối: CaCO3 -> CaO + CO2↑ (t°)
- Nhiệt phân bazơ không tan: Cu(OH)2 -> CuO + H2O (t°)

### 2. Điều chế axit

- Phi kim + hiđro: H2 + Cl2 -> 2HCl (t° hoặc ánh sáng)
- Oxit axit + nước: SO3 + H2O -> H2SO4
- Axit mạnh, khó bay hơi + muối (phương pháp sunfat): H2SO4 (đặc) + 2NaCl (khan) -> Na2SO4 + 2HCl↑ (t°)
- Sản xuất axit sunfuric trong công nghiệp theo sơ đồ: FeS2 -> SO2 -> SO3 -> H2SO4.

### 3. Điều chế bazơ

- Oxit bazơ + nước: BaO + H2O -> Ba(OH)2
- Kiềm + dung dịch muối: Ca(OH)2 + K2CO3 -> 2KOH + CaCO3↓; 2KOH + FeSO4 -> Fe(OH)2↓ + K2SO4
- Điện phân dung dịch muối ăn bão hoà (có màng ngăn xốp): 2NaCl + 2H2O -> 2NaOH + Cl2↑ + H2↑ (điện phân có màng ngăn)

### 4. Điều chế muối

Từ hợp chất:

- Axit + bazơ: HCl + NaOH -> NaCl + H2O
- Axit + oxit bazơ: H2SO4 + CuO -> CuSO4 + H2O
- Oxit axit + dung dịch bazơ: SO3 + 2NaOH -> Na2SO4 + H2O
- Oxit axit + oxit bazơ: CO2 + CaO -> CaCO3
- Dung dịch muối + dung dịch muối: Na2SO4 + BaCl2 -> 2NaCl + BaSO4↓
- Dung dịch bazơ + dung dịch muối: 2KOH + MgCl2 -> 2KCl + Mg(OH)2↓
- Dung dịch muối + axit: Na2CO3 + 2HCl -> 2NaCl + H2O + CO2↑

Từ đơn chất:

- Kim loại + phi kim: 2Na + Cl2 -> 2NaCl (t°)
- Kim loại + axit: Fe + 2HCl -> FeCl2 + H2↑
- Kim loại + dung dịch muối: Fe + CuCl2 -> FeCl2 + Cu↓
